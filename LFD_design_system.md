# LFD Design System

LFD의 공식 디자인 시스템 가이드라인 문서입니다. 본 문서는 브랜드 고유의 컬러 시스템, 그라디언트 시스템, 백그라운드 색상 활용 및 그레이 스케일 가이드를 정의합니다.

---

## 1. Color System

LFD의 브랜드 컬러 시스템은 하나의 핵심 색상을 기준으로 다양한 명도와 채도의 단계별 셰이드(Shades)를 제공하여 일관된 시각적 흐름을 구축합니다.

| 단계 (Level) | 색상명 (Name) | HEX Code | 주요 용도 (Usage) |
|:---|:---|:---|:---|
| **00** | LFD Cyan | `#04CDF8` | 핵심 포인트 컬러, 액티브 하이라이트, 액션 뱃지 |
| **20** | LFD Blue | `#0071E3` | 주요 대화형 버튼, 하이퍼링크, 프라이머리 텍스트 링크 |
| **40** | LFD Royal Blue | `#0033CC` | 버튼 마우스 오버(Hover) 상태, 포커스 하이라이트 |
| **60** | LFD Navy | `#001168` | 베이스 그라디언트 시작점, 컴포넌트 내부 강조 영역 |
| **80** | LFD Dark Navy | `#00052A` | 메인 백그라운드 레이어, 베이스 컴포넌트 배경 |
| **100** | LFD Black | `#000000` | 메인 상단 헤더, 모달 백드롭, 베이스 그림자 영역 |

---

## 2. Gradient System

LFD의 시그니처 그라디언트는 극도의 모던함과 고급스러움을 연출하기 위해 **LFD Black (100)**에서 **LFD Navy (60)** 및 **LFD Dark Navy (80)**로 점진적으로 흐르는 어두운 심해 느낌의 그라디언트를 사용합니다.

- **메인 캔버스 그라디언트**:
  - 시작 색상: `LFD Black (#000000)`
  - 중간 색상: `LFD Dark Navy (#00052A)`
  - 끝 색상: `LFD Navy (#001168)`
  - 각도: 180도 (위에서 아래로)

## 3. Background Color Usage

배경 색상에 따른 로고 및 텍스트의 활용 규칙입니다.

1. **그라디언트 및 어두운 배경 (LFD 100 ~ 60)**:
   - 로고 마크는 **단색 화이트** 사용을 원칙으로 합니다.
   - 텍스트는 화이트 및 실버 그레이를 적용하여 높은 명도 대비와 가독성을 보장합니다.
2. **밝은 컴포넌트 배경 (LFD 20 ~ 00)**:
   - 로고 마크는 상황에 따라 **단색 화이트** 또는 **단색 블랙**을 사용합니다.

---

## 4. Gray Scale

어두운 모드에서 텍스트 및 UI 요소의 계층을 구축하기 위한 회색조 시스템입니다.

- **90**: `#1C1C1E` (다크 그레이 1)
- **80**: `#2C2C2E` (다크 그레이 2)
- **70**: `#3A3A3C` (미디엄 다크 그레이)
- **60**: `#48484A` (보조 다크 그레이)
- **50**: `#636366` (그레이)
- **40**: `#8E8E93` (보조 텍스트 그레이)
- **30**: `#AEAEB2` (플레이스홀더 그레이)
- **20**: `#C7C7CC` (실버 그레이)
- **10**: `#E5E5EA` (오프 화이트)
- **00**: `#FFFFFF` (순수 화이트)
